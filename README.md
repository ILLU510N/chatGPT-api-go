# ChatGPT api for golang

simple demo for golang chatGPT api server.

---

powered by gin. 

### Install
```sh
git clone https://github.com/ILLU510N/chatGPT-api-go.git
go build
./chatgpt-api-go
```

### Config
conf.yaml

```yaml
# example
port: 8080
# use proxy to visit openai
proxy: http://127.0.0.1:1234
api_key: yourkey
```
